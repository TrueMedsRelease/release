Häufige Anwendung

Ciclosporin wird zur Vorbeugung von Organabstoßungen nach einer Nieren-, Herz- oder Lebertransplantation eingesetzt. Ciclosporin schwächt das körpereigene Immunsystem und verhindert so die Abstoßung eines transplantierten Organs, z. B. einer Niere. Eine Organabstoßung tritt auf, wenn das Immunsystem das neue Organ als Eindringling betrachtet und angreift. Ciclosporin wird auch zur Behandlung von schwerer Psoriasis oder schwerer rheumatoider Arthritis eingesetzt.

Dosierung und Anwendungsgebiet

Wenden Sie das Arzneimittel genau nach Anweisung an. Ciclosporin oral wird oral eingenommen. Sie können Ciclosporin oral mit oder ohne Nahrung einnehmen, die Einnahme sollte jedoch jedes Mal auf die gleiche Weise erfolgen. Nehmen Sie das Arzneimittel jeden Tag zur gleichen Zeit ein.

Vorsichtsmaßnahmen

Informieren Sie Ihren Arzt, wenn Sie bereits eine der folgenden Erkrankungen hatten: Leber- oder Nierenerkrankung; Bluthochdruck; oder wenn bei Ihnen eine Impfung geplant ist.
Es ist nicht bekannt, ob dieses Arzneimittel einem ungeborenen Kind schadet. Informieren Sie Ihren Arzt, wenn Sie schwanger sind oder eine Schwangerschaft planen. Sie sollten während der Anwendung von Ciclosporin nicht stillen.
Grapefruit kann Wechselwirkungen mit Ciclosporin haben und zu unerwünschten Nebenwirkungen führen. Vermeiden Sie den Verzehr von Grapefruitprodukten.
Vermeiden Sie Alkohol.
Vermeiden Sie den Kontakt mit kranken oder infizierten Personen. Informieren Sie Ihren Arzt sofort, wenn Sie Anzeichen einer Infektion entwickeln.

Kontraindikationen

Sie sollten Ciclosporin nicht anwenden, wenn Sie allergisch darauf reagieren. Sie können Ciclosporin möglicherweise nicht anwenden, wenn Sie an einer Nierenerkrankung, unbehandeltem oder unkontrolliertem Bluthochdruck oder einer Krebserkrankung leiden.
Wenn Sie wegen Psoriasis behandelt werden, sollten Sie während der Behandlung mit Ciclosporin keine UV-Lichttherapie (PUVA oder UVB), keine Strahlentherapie, keine Kohlenteertherapie oder Medikamente erhalten, die das Immunsystem schwächen (wie z. B. Methotrexat).

Mögliche Nebenwirkungen

Ciclosporin kann schwerwiegende Nebenwirkungen verursachen. Wenden Sie sich sofort an Ihren Arzt, wenn Sie Folgendes bemerken: Probleme mit Sprache, Denken, Sehen oder Muskelbewegungen (können schleichend beginnen und sich schnell verschlimmern); einen Krampfanfall; starke Kopfschmerzen, verschwommenes Sehen, Pochen im Nacken oder in den Ohren; blasse Haut, leichte Blutergüsse, ungewöhnliche Blutungen; einen hohen Kaliumspiegel – Übelkeit, Schwäche, Kribbeln, Brustschmerzen, unregelmäßiger Herzschlag, Bewegungseinschränkung; Nierenprobleme; Leberprobleme; oder Anzeichen einer Infektion.
Häufige Nebenwirkungen von Ciclosporin können sein: Zittern oder Schütteln; Akne, verstärkter Gesichts- oder Körperbehaarung; erhöhter Blutdruck; Übelkeit, Durchfall; oder geschwollenes oder schmerzendes Zahnfleisch.

Wechselwirkungen mit anderen Arzneimitteln

Ciclosporin kann Ihre Nieren schädigen, insbesondere wenn Sie gleichzeitig bestimmte Arzneimittel gegen Infektionen, Krebs, Osteoporose, Transplantatabstoßung, Darmerkrankungen, Bluthochdruck, Schmerzen oder Arthritis (einschließlich Advil, Motrin und Aleve) einnehmen.

Verpasste Dosis

Nehmen Sie das Arzneimittel so bald wie möglich ein. Lassen Sie die vergessene Dosis jedoch aus, wenn es fast Zeit für die nächste Dosis ist. Nehmen Sie nicht zwei Dosen gleichzeitig ein.

Überdosierung

Suchen Sie sofort einen Arzt auf.

Aufbewahrung

Bei Raumtemperatur vor Feuchtigkeit und Hitze schützen. Außerhalb der Reichweite von Kindern aufbewahren.

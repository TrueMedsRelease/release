Häufige Anwendung

Glucophage Trio Tabletten werden zur Behandlung von Typ-2-Diabetes mellitus angewendet, wenn Diät, Bewegung und einzelne Antidiabetika den Blutzuckerspiegel nicht ausreichend kontrollieren können.

Dosierung und Anwendungsgebiet

Nehmen Sie Glucophage Trio Tabletten nach Anweisung Ihres Arztes ein. Schlucken Sie die Tabletten unzerkaut mit einem Glas Wasser. Zerteilen, kauen oder zerbrechen Sie das Arzneimittel nicht. Für optimale Ergebnisse nehmen Sie die Tabletten regelmäßig ein. Nehmen Sie nicht mehr als die von Ihrem Arzt verordnete Dosis ein.

Vorsichtsmaßnahmen

Sprechen Sie mit Ihrem Arzt, wenn: Sie in der Vergangenheit an Durchfall gelitten haben; Sie blutdrucksenkende Medikamente, Schmerzmittel oder Wassertabletten einnehmen; Sie bereits eine Laparotomie (chirurgischer Eingriff in die Bauchhöhle) hatten; Sie leiden an schweren Darmerkrankungen wie dem Roemheld-Syndrom (Druck im Magen-Darm-Trakt), einer schweren Hernie und Stenose (abnorme Verengung eines Blutgefäßes) oder einem Geschwür im Darm. Sie haben eine Nieren- oder Lebererkrankung.

Kontraindikationen

Wenn Sie allergisch gegen Metformin, Glimepirid, Voglibose oder einen der sonstigen Bestandteile von Glucophage Trio Tabletten sind. Wenn Sie schwanger sind oder stillen. Wenn Sie eine Nieren- oder Lebererkrankung haben. Wenn Sie an einer akuten metabolischen Azidose leiden (bei der der Körper zu viel organische Säuren produziert). Wenn Sie in der Vergangenheit aufgrund unzureichender Sauerstoffversorgung an einer Herzerkrankung wie Schock, Herzinsuffizienz usw. gelitten haben. Wenn Sie Alkoholiker sind. Wenn Sie vor oder nach einer Operation eine schwere Infektion hatten. Wenn Sie an einer Magen-Darm-Erkrankung leiden.

Mögliche Nebenwirkungen

Nebenwirkungen von Glucophage Trio: Kopfschmerzen, Schwindel, Übelkeit, Erbrechen, Durchfall.

Wechselwirkungen mit anderen Arzneimitteln

Besondere Vorsicht ist geboten bei der Einnahme von Arzneimitteln zur Behandlung von Tuberkulose (z. B. Rifampicin), von Infektionen (z. B. Fluconazol), von Bluthochdruck (ACE-Hemmer wie Captopril, Atenolol), von Steroiden (z. B. Chloramphenicol), Fenfluramin, Fenyramidol), von cholesterinsenkenden Arzneimitteln (Fibraten), von Medikamenten gegen Depressionen (Fluoxetin), von Blutgerinnungshemmern (z. B. Cumarine), von Cyclophosphamid, Miconazol, Probenecid oder von Antibiotika (z. B. Clarithromycin, Doxycyclin, Ofloxacin). und Antiallergika (Ranitidin), Clonidin und Reserpin.
Wenn Sie andere Diabetesmedikamente wie Insulin und Pioglitazon einnehmen, können Symptome eines niedrigen Blutzuckerspiegels auftreten. Zusätzlich können Symptome eines hohen Blutzuckerspiegels auftreten, wenn Sie orale Kontrazeptiva, Acetazolamid, Diuretika, Phenytoin, Rifampicin, Medikamente gegen Verstopfung oder Medikamente gegen Schilddrüsenerkrankungen einnehmen. Medikamente wie Diclofenac und Ibuprofen können Ihre Nierenfunktion beeinträchtigen, wenn sie zusammen mit diesem Arzneimittel eingenommen werden.
Glucophage Trio Tabletten können die Wirkung von gerinnungshemmenden Medikamenten wie Phenprocoumon verändern.

Verpasste Dosis

Wenn Sie eine Dosis Glucophage Trio vergessen haben, holen Sie diese nach, sobald Sie sich daran erinnern. Wenn es Zeit für die nächste Dosis ist, lassen Sie die vergessene Dosis aus und setzen Sie Ihre gewohnte Einnahme fort. Nehmen Sie nicht die doppelte Dosis ein, um eine vergessene Einnahme nachzuholen.

Überdosierung

Eine Überdosierung von Glucophage Trio kann zu Unterzuckerung, Schwäche, Zittern und Übelkeit führen. Suchen Sie sofort einen Arzt auf, wenn Sie glauben, zu viel von diesem Arzneimittel eingenommen zu haben.

Aufbewahrung

Lagern Sie Glucophage Trio Tabletten unter 25 °C. Bewahren Sie das Arzneimittel außerhalb der Reichweite von Kindern und Haustieren auf.
